# Create an Invoice

Create professional invoices that get paid quickly. This guide walks through creating an invoice from start to finish.

---

## Before You Start

Have ready:
- [ ] Client name or project
- [ ] Work performed or milestone reached
- [ ] Agreed rates or prices
- [ ] Any credits or adjustments

---

## Method 1: Invoice from Project

Best for: Progress billing, milestone payments

### Step 1: Open the Project

1. Go to **Projects**
2. Find and open the project
3. Click **Financials** tab (or **Create Invoice** button)

### Step 2: Click Create Invoice

1. Click **Create Invoice**
2. System pre-fills project and client info

### Step 3: Choose What to Bill

**Percentage Complete:**
- Enter completion percentage
- System calculates amount based on contract value
- Good for: Progress billing

**Specific Items:**
- Add line items manually
- Reference phases or work completed
- Good for: Milestone billing

**From Time Entries:**
<!-- STATUS: COMING_SOON - Time-based invoicing -->
- Select time entries to bill
- System calculates based on rates
- Good for: T&M work

### Step 4: Review and Add Details

1. Check all line items are correct
2. Add descriptions as needed
3. Verify pricing
4. Review total

### Step 5: Set Due Date

Select payment terms:
- Due on Receipt
- Net 15
- Net 30
- Net 45
- Custom date

### Step 6: Add Notes (Optional)

Include:
- Thank you message
- What's coming next
- Contact for questions

### Step 7: Send

1. Click **Send Invoice**
2. Verify client email
3. Add email message (optional)
4. Send

---

## Method 2: Create from Scratch

Best for: One-off billing, non-project work

### Step 1: Start New Invoice

1. Go to **Invoicing**
2. Click **New Invoice**

### Step 2: Select Client

1. Search for existing client
2. Or add new client inline

### Step 3: Link to Project (Optional)

1. If related to a project, select it
2. Helps with reporting and tracking
3. Skip if standalone invoice

### Step 4: Add Line Items

For each item:
1. Click **Add Line Item**
2. Enter description
3. Set quantity and price
4. Review line total

**Tip:** Be specific in descriptions so client knows exactly what they're paying for.

### Step 5: Review Summary

Check:
- Subtotal correct
- Tax applied if needed
- Total matches expected

### Step 6: Set Terms

1. Choose due date
2. Add payment instructions
3. Include any terms

### Step 7: Send

1. Preview the invoice
2. Click **Send**
3. Invoice emailed to client

---

## Adding Line Items

### From Contract/Estimate

If you have an approved estimate:
1. Click **Import from Estimate**
2. Select the estimate
3. Choose which items to include
4. Adjust quantities if partial billing

### From Phases

Reference project phases:
1. Click **Add from Phase**
2. Select completed phase
3. Amount pulls from phase budget

### Manual Entry

Enter custom items:

| Field | Example |
|-------|---------|
| Description | Demo and removal of existing kitchen cabinets |
| Quantity | 1 |
| Unit | LS |
| Rate | $1,500.00 |
| Total | $1,500.00 |

### Time-Based Items

<!-- STATUS: COMING_SOON - Time import -->

Import from timesheets:
1. Click **Add from Time**
2. Select date range
3. Choose employees to include
4. Review rates
5. Add to invoice

## Invoice Details to Include

### Required Information

- Invoice number (auto-generated)
- Invoice date
- Due date
- Client name and address
- Your company info
- Line items with amounts
- Total amount due

### Recommended Details

- Project name/address
- PO number (if client requires)
- Reference to contract or estimate
- Payment instructions
- Late payment terms

### Optional Additions

- Phase or milestone name
- Detailed work descriptions
- Attached documentation
- Personal thank you note

---

## Review Before Sending

### Checklist

- [ ] Client name spelled correctly
- [ ] Billing address accurate
- [ ] All work included
- [ ] Amounts are correct
- [ ] Tax applied correctly (if applicable)
- [ ] Due date reasonable
- [ ] Payment instructions clear
- [ ] No typos or errors

### Preview

1. Click **Preview**
2. See exactly what client will receive
3. Check formatting and layout
4. Verify PDF looks professional

---

## Sending Options

### Email (Recommended)

1. Click **Send Invoice**
2. Email auto-populates from client record
3. Add personalized message
4. Send

**Email includes:**
- Professional message
- PDF attachment
- Pay Online button (if enabled)

### Download PDF

1. Click **Download PDF**
2. Save to your computer
3. Email or print manually
4. Mark as sent in system

### Print

1. Click **Print**
2. Opens PDF in browser
3. Print to paper
4. Mark as sent in system

---

## After Sending

### Track Status

Watch for:
- **Sent** — Email delivered
- **Viewed** — Client opened it
- **Paid** — Payment received

### Follow Up

If no payment after due date:
1. Send reminder
2. Call if needed
3. Adjust terms for future

### Payment Received

When paid:
1. Mark as paid (auto if online payment)
2. Or record manual payment
3. Invoice closes

---

## Common Scenarios

### Progress Invoice (Percentage)

Billing for 25% completion:

1. Contract value: $40,000
2. Create invoice
3. Select "Percentage Complete"
4. Enter 25%
5. Invoice amount: $10,000

### Milestone Invoice

Billing for rough-in complete:

1. Create invoice
2. Add line item: "Rough-in phase complete per contract"
3. Amount per contract milestone
4. Reference contract section

### Change Order Invoice

Billing for approved change:

1. Create invoice
2. Add line item: "Change Order #3 - Additional electrical outlet"
3. Amount per signed CO
4. Attach CO document

### Final Invoice

Billing project completion:

1. Create invoice
2. Add final items
3. Subtract previous payments
4. Balance due = remaining contract amount

## Troubleshooting

### Invoice won't send

- Check client email is valid
- Verify email settings configured
- Check spam folder (ask client)

### Amounts don't match contract

- Review estimate/contract
- Check for change orders
- Verify previous invoices
- Adjust line items

### Client can't pay online

- Verify payment processor connected
- Check invoice has pay button
- Resend invoice
- Provide manual payment option
