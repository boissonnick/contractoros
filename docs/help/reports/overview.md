## Running Reports

### Selecting a Report

1. Go to **Reports**
2. Choose category (Projects, Financial, etc.)
3. Select specific report
4. Report loads with default settings

### Filtering & Customizing

Most reports allow filtering by:
- Date range
- Project(s)
- Client(s)
- Employee(s)
- Status

### Grouping & Sorting

Organize data by:
- Ascending/descending
- Group by category
- Show/hide columns
- Subtotals and totals

### Exporting

Export reports in multiple formats:
- **PDF** — For printing and sharing
- **Excel** — For further analysis
- **CSV** — For importing elsewhere

## Report Best Practices

### Regular Review

Set a routine:
- **Daily:** Dashboard check
- **Weekly:** AR aging, project status
- **Monthly:** Financial summary, time analysis
- **Quarterly:** Profitability, trends
- **Annually:** Full business review

### Compare Periods

Look for trends:
- This month vs last month
- This quarter vs same quarter last year
- Year over year growth

### Take Action

Reports should drive decisions:
- AR aging → Collection calls
- Project overruns → Process improvements
- Sub performance → Vendor management
- Client concentration → Diversification

### Share Appropriately

Who should see what:
- Owners: Everything
- PMs: Project-level reports
- Team: Their own time data
- Clients: Their project reports only

## Common Questions

### Can I save favorite reports?

Yes. Click the star icon on any report to save it to favorites for quick access.

### How current is the data?

Reports use real-time data. What you see reflects current state.

### Can I share reports externally?

Export as PDF to share via email. For client-facing reports, use the Client Portal.

### Why don't my numbers match my accounting software?

<!-- STATUS: NEEDS_SCOPE - Accounting sync -->
ContractorOS tracks project data. Some items may differ based on timing, methodology, or what's entered in each system.

### Can I get reports on mobile?

Basic dashboard is available. For full reports, use a desktop browser for best experience.
