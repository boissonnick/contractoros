## Submitting Expenses

### From the Field (Mobile)

1. Open Field Portal
2. Tap **Add Expense**
3. Take photo of receipt
4. Enter amount
5. Select category
6. Choose project
7. Add description
8. Submit

### From the Dashboard

1. Go to **Expenses**
2. Click **New Expense**
3. Enter details:
   - Amount
   - Date
   - Category
   - Project (if applicable)
   - Vendor
   - Description
4. Attach receipt
5. Submit

→ [Submit an expense](/help/expenses/submit-expense.guide.md)

## Receipt Management

### Why Receipts Matter

- Required for reimbursement
- Tax documentation
- Audit protection
- Cost verification

### Capturing Receipts

**Best method:** Photo from phone

1. Take photo immediately
2. Ensure text is readable
3. Capture full receipt
4. Good lighting helps

**Alternative:** Upload later
1. Scan or photograph
2. Attach to expense entry
3. Keep originals as backup

### No Receipt?

If receipt is lost:
1. Document what happened
2. Include as much detail as possible
3. May require manager approval
4. Consider policy for future

## Approval Workflow

### Submission

```
Submit → Review → Approve/Reject → Reimburse
                       ↓
                 Request Changes
```

### Review Process

Managers review expenses for:
- Valid business purpose
- Correct project assignment
- Receipt attached
- Amount accuracy
- Policy compliance

### Approval Actions

| Action | When to Use |
|--------|-------------|
| Approve | Expense is valid and documented |
| Reject | Not a valid business expense |
| Request Changes | Need more info or corrections |

### Notifications

- Submitter notified of decision
- Approved expenses queue for payment
- Rejected expenses returned with reason

## Reimbursements

### Getting Reimbursed

Approved employee expenses can be reimbursed via:
- **Payroll** — Added to next paycheck
- **Direct payment** — Separate reimbursement

### Reimbursement Process

1. Submit expense with receipt
2. Manager approves
3. Included in payroll or separate payment
4. Status updates to "Reimbursed"

### Tracking Your Expenses

As an employee, view your expenses:
- Pending (awaiting approval)
- Approved (reimbursement pending)
- Reimbursed (paid)
- Rejected (returned)

## Best Practices

### For Employees

| Do | Don't |
|----|-------|
| Submit immediately | Wait until month-end |
| Always attach receipt | Submit without documentation |
| Be specific in descriptions | Use vague descriptions |
| Select correct project | Leave project blank |

### For Managers

| Do | Don't |
|----|-------|
| Review promptly | Let expenses pile up |
| Check receipts | Approve without reviewing |
| Ask questions | Assume or guess |
| Be consistent | Apply rules unevenly |

### For the Business

1. **Clear policy** — What's allowed, what's not
2. **Spending limits** — Who can approve what
3. **Regular review** — Weekly expense review
4. **Timely reimbursement** — Don't delay payments

## Common Questions

### How long do I have to submit expenses?

Submit within 30 days (or per your company policy). Older expenses may require additional approval.

### What if I don't have a receipt?

Submit with as much detail as possible and note that receipt is unavailable. Your manager may require additional documentation.

### Can I submit personal expenses?

Only if they're for legitimate business purposes. Personal expenses not related to work should not be submitted.

### How are expenses charged to projects?

When you select a project during submission, the expense is associated with that project for cost tracking.

### What happens to rejected expenses?

You're notified with a reason. You can correct and resubmit or discuss with your manager.
