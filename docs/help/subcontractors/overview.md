# Subcontractor Management Overview

Build and manage your network of trusted subcontractors. Send bid requests, track their work, manage documents, and streamline payments.

---

## What You'll Learn

| Job to Be Done | This Section Helps You |
|----------------|------------------------|
| Build a sub database | Create and organize sub profiles |
| Get competitive bids | Send bid requests to multiple subs |
| Assign subs to projects | Track who's working where |
| Verify compliance | Manage insurance and licenses |
| Pay subs accurately | Track work and generate payments |

---

## The Sub Lifecycle

```
Add Sub → Verify → Bid → Award → Work → Pay
              ↓
          Decline/Archive
```

### 1. Add Sub
- Create profile with contact info
- Note their specialty trades
- Add to your network

### 2. Verify
- Collect insurance certificates
- Verify licenses
- Check references
- Mark as approved

### 3. Bid
- Send bid requests for projects
- Receive and compare bids
- Negotiate terms

### 4. Award
- Select winning bid
- Notify sub
- Generate subcontract

### 5. Work
- Sub performs work
- Track progress
- Document completion

### 6. Pay
- Receive invoices
- Approve for payment
- Process payments

## Adding Subcontractors

### Quick Add

1. Go to **Subcontractors**
2. Click **Add Subcontractor**
3. Enter basic info:
   - Company name
   - Contact name
   - Phone and email
   - Primary trade
4. Save

### Full Profile

After quick add, complete the profile:

1. Open the sub's profile
2. Add:
   - Multiple contacts
   - Service areas
   - All trades they perform
   - Hourly/daily rates
   - Notes and preferences
3. Request compliance documents

→ [Add a subcontractor](/help/subcontractors/add-sub.guide.md)

## Trades & Specialties

### Organizing by Trade

Categorize subs by their specialties:

| Trade Category | Examples |
|----------------|----------|
| Sitework | Excavation, grading, concrete |
| Structural | Framing, steel, masonry |
| MEP | Plumbing, electrical, HVAC |
| Finishes | Drywall, paint, flooring, tile |
| Specialty | Roofing, windows, insulation |

### Multi-Trade Subs

Many subs do multiple trades:
- Mark all applicable trades
- Primary trade for main categorization
- Searchable by any trade

### Trade Preferences

Set preferences per trade:
- Preferred subs (get first call)
- Approved subs (can bid)
- Do not use (past issues)

## Compliance Tracking

### Required Documents

| Document | Why It's Needed |
|----------|-----------------|
| Certificate of Insurance (COI) | Protects you from liability |
| Workers' Comp Certificate | Required in most states |
| General Liability Certificate | Covers property damage |
| License Copy | Verifies legal operation |
| W-9 | Required for 1099 reporting |

### Expiration Tracking

The system tracks expirations:
- Dashboard shows expiring soon
- Automatic reminders to subs
- Block from bidding if expired
- Color-coded status

### Collecting Documents

**Method 1: Sub Portal**
1. Invite sub to portal
2. They upload documents directly
3. You verify and approve

**Method 2: Manual Upload**
1. Receive documents via email
2. Upload to sub profile
3. Set expiration dates

→ [Manage compliance](/help/subcontractors/compliance.guide.md)

## Bid Requests

### Creating a Bid Request

1. Go to **Bid Requests** (or from a project)
2. Click **New Bid Request**
3. Select trade needed
4. Add scope description
5. Attach drawings/specs
6. Set due date
7. Select subs to invite
8. Send

### What Subs Receive

- Email notification
- Link to bid request
- Scope details and attachments
- Bid submission form
- Due date

### Receiving Bids

As subs respond:
- Bids appear in system
- Compare side by side
- See price, timeline, notes
- Evaluate and select

### Awarding Work

1. Select winning bid
2. Click **Award**
3. Sub is notified
4. Generate subcontract (optional)
5. Losing bidders notified (optional)

→ [Send bid requests](/help/subcontractors/bid-requests.guide.md)

## Assigning Subs to Projects

### From the Project

1. Open project
2. Go to **Team** tab
3. Click **Add Subcontractor**
4. Search and select sub
5. Assign to phase(s)
6. Set schedule dates

### From Sub Profile

1. Open sub's profile
2. Click **Assign to Project**
3. Select project
4. Choose phases
5. Confirm

### What Happens

- Sub appears on project team
- Gets access to project (if portal enabled)
- Can see their assigned work
- Can log time and progress

## Sub Payments

### Payment Workflow

```
Sub Invoice → Review → Approve → Pay
                 ↓
              Request Changes
```

### Receiving Invoices

**Method 1: Sub Portal**
- Sub submits invoice online
- Automatically linked to project
- You review and approve

**Method 2: Manual Entry**
- Receive paper/email invoice
- Enter into system
- Link to project and work

### Reviewing Invoices

Before approving, verify:
- Work was completed
- Amount matches agreement
- Backup documentation
- No disputes

### Approving for Payment

1. Review invoice
2. Click **Approve**
3. Invoice moves to payables
4. Generate payment when ready

### Retention

<!-- STATUS: NEEDS_SCOPE - Retention tracking -->

Handle retention automatically:
- Set retention percentage (typically 10%)
- Hold amount from each payment
- Release on final completion

## Best Practices

### Building Your Network

1. **Verify early** — Get documents before you need them
2. **Multiple per trade** — Have 3+ subs for each trade
3. **Regular updates** — Keep contact info current
4. **Performance notes** — Document issues and successes

### Bidding Success

1. **Clear scope** — Detailed scope gets accurate bids
2. **Fair timelines** — Realistic bid due dates
3. **Respond quickly** — Subs appreciate prompt decisions
4. **Negotiate fairly** — Build long-term relationships

### Maintaining Relationships

1. **Pay on time** — Nothing damages trust faster
2. **Clear communication** — Set expectations upfront
3. **Fair treatment** — Handle disputes professionally
4. **Recognition** — Acknowledge good work

## Common Questions

### Can subs see my other projects?

No. Subs only see projects they're assigned to.

### How do I remove a sub from a project?

Go to project team, find the sub, click remove. Their work history stays.

### What if a sub's insurance expires?

You're notified. They're flagged in the system. You can block them from new work until renewed.

### Can subs submit their own invoices?

Yes, through the Sub Portal. You review and approve.

### How do I track retention?

Set retention percentage in settings. System calculates and tracks automatically.
